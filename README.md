# Hand-Tracking
## Development environment
 * Windows 10
 * C++14
 * Opencv 4.5.1
 * Visual Studio 2019
 
## Execution
### Requirement
  * Opencv 4.5.1
  * g++ compiler to c++14

### Run
In windows

   `git clone https://github.com/MyllenaAPrado/Hand-Tracking.git`
   
   `cd Hand-Tracking/HandTracking/HandTracking`
   
   `g++ HandTracking.cpp -o HandTracking -Wall`
   
   `./HandTracking`
